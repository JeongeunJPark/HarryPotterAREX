All prefabs are preset.
Under prefabs folder locate the lightshower prefab and set its transform rotation x to "-90"
Under prefabs folder locate the rainingheavily prefab and set its transform rotation y to "-90"
Under prefabs folder locate the tornado prefab and set its transform rotation x to "-90"
Under prefabs folder locate the Sun prefab and set its transform rotation x to "-90"
Demo scene is implemented to test the particles.
In order to change the direction fo fog change the velocity over lifetime attribute of the particle system.
To change the size of any prefab change the start size attribute of the particle system.
